# .NET Core Internship - Week 10: Dependency Injection & Middleware
## Developer: EZZALDEEN

### 📝 About the Project
This project represents the work completed during Week 10 of the intensive .NET Core Internship. It focuses on implementing advanced Web API concepts, emphasizing clean software architecture and actual SQL Server database integration.

### 🛠 Tech Stack
*   **Framework:** .NET 10.0.300
*   **Database:** SQL Server 2022
*   **Language:** C# (SOLID Principles)
*   **Libraries:** Native Microsoft Libraries (ADO.NET, Microsoft.Data.SqlClient)

### 🌟 Key Features
*   **Books API (Weekly Assignment):**
    *   Full refactoring using the **Service Layer Pattern**.
    *   Actual integration with SQL Server for complete CRUD operations.
    *   Implementation of **Health Checks** for database connectivity monitoring.
*   **Request Logger (Mini Project):**
    *   Professional **Custom Middleware** to log all requests in a structured JSON format.
    *   Automated **Slow Request Alert** system for performance monitoring.
*   **Advanced Middleware Pipeline:**
    *   **Correlation ID:** Tracking requests with unique identifiers.
    *   **Performance Timing:** Measuring execution time and injecting it into response headers.
    *   **Logging:** Comprehensive documentation of request paths and statuses.
*   **DI Lifetime Demo:**
    *   Practical implementations demonstrating the differences between **Transient**, **Scoped**, and **Singleton** lifetimes.

### 📁 Project Structure
*   `WeeklyAssignment_BooksAPI/`: The core project for book management.
*   `MiniProject_RequestLogger/`: A professional request logging API.
*   `Deliverables_Page15/`: Contains independent tasks (Greeting Service, Email Swap, DI Demo).
*   `Database_Setup.sql`: SQL script for database, tables, and seed data creation.
*   `PROJECT_STANDARDS.md`: Technical standards and guidelines followed during development.
